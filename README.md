# Todo Calendar - Android待办事项应用

一个功能完整的Android待办事项应用，与系统日历深度集成，支持持久化提醒。

## ✨ 主要功能

- **📝 任务管理**
  - 添加新任务到日历
  - 编辑现有任务
  - 删除任务
  - 标记任务完成/未完成
  
- **📅 日历集成**
  - 所有任务自动同步到系统日历
  - 即使应用被关闭，系统日历仍会发送提醒
  - 任务的编辑和删除自动同步到日历

- **🔔 智能提醒**
  - 自定义提醒时间（1-60分钟提前）
  - 使用系统日历提醒机制，确保可靠性
  - 即使应用被Kill，提醒依然有效

- **🎨 现代化UI**
  - 使用Jetpack Compose构建
  - Material Design 3设计语言
  - 流畅的动画和交互

## 🛠 技术栈

- **语言**: Kotlin
- **UI框架**: Jetpack Compose
- **架构**: Single Activity
- **最低SDK**: Android 8.0 (API 26)
- **目标SDK**: Android 14 (API 34)

### 核心技术

- Jetpack Compose - 现代化声明式UI
- Material 3 - 最新设计系统
- Calendar Provider API - 系统日历集成
- Runtime Permissions - 动态权限管理
- ContentResolver - 内容提供器操作

## 📋 权限说明

应用需要以下权限：

- `READ_CALENDAR` - 读取日历事件以显示任务列表
- `WRITE_CALENDAR` - 创建、编辑和删除日历事件

## 🚀 功能特点

### 1. 添加任务
- 输入任务标题（必填）
- 添加任务描述（可选）
- 设置提醒时间（1-60分钟）
- 任务自动添加到系统日历

### 2. 编辑任务
- 修改任务标题和描述
- 调整提醒时间
- 更改自动同步到日历

### 3. 删除任务
- 一键删除任务
- 同时从日历中移除

### 4. 任务状态
- 勾选框标记完成状态
- 视觉反馈显示已完成任务

## 📱 核心功能实现

### Calendar Provider集成

应用使用Android的Calendar Provider API来实现与系统日历的双向同步：

- **添加事件**: 使用`CalendarContract.Events`创建事件
- **添加提醒**: 使用`CalendarContract.Reminders`设置提醒
- **查询事件**: 从日历读取所有未来的事件
- **更新事件**: 修改现有事件的标题和描述
- **删除事件**: 从日历中移除事件

### 为什么选择日历集成？

1. **可靠的提醒**: 系统日历提醒由Android系统管理，即使应用被关闭或Kill，提醒依然有效
2. **跨应用访问**: 用户可以在任何日历应用中查看和管理任务
3. **系统级持久化**: 数据存储在系统数据库中，更加可靠
4. **原生通知**: 使用系统原生的提醒通知，用户体验更好

## 📝 使用说明

1. **首次使用**: 应用会请求日历权限，点击"授予权限"
2. **添加任务**: 点击右下角的浮动按钮 (+)
3. **编辑任务**: 点击任务卡片上的编辑图标
4. **删除任务**: 点击任务卡片上的删除图标
5. **标记完成**: 点击任务卡片左侧的复选框
